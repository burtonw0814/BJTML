# BJTML

 Extension of JTML software to be compatible with DU's biplanar tracking workflow.
 
 Single plane version:
 https://github.com/ajensen1234/JTML/

 Instructions:
 1. Install dependencies:
    Follow instructions at the following link to install dependencies:
    https://github.com/burtonw0814/BJTML_link/blob/main/LINUX_BUILD.md
 2. Build using CMake:
    cmake --build .
    cmake CMakeLists.txt
 3. Run:
    ./Biplanar-Joint-Track-Machine-Learning

 Other notes:
 1. Camera calibration data is based on calibration data exported from XMA Lab.
    Format XMA Lab-based camcal data using code in the MATLAB directory.
